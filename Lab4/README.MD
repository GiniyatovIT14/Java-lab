# Проект Java/Лабораторная работа №4. Обобщенные типы

## Описание проекта
Проект представляет собой комплексную систему, реализующую различные аспекты объектно-ориентированного программирования на Java. 
Включает систему обобщенных контейнеров, функциональных интерфейсов, сравнения объектов и работы с коллекциями.
Все задачи объединены в единую систему с интерактивным меню для демонстрации функциональности.

## Структура проекта

### Основные классы:
```
- Main.java              - главный класс программы с системой меню
- Box.java               - обобщенный контейнер для хранения объектов
- NumberValue.java       - числовое значение с поддержкой сравнения
- MathUtils.java         - утилиты для математических операций
- ListUtils.java         - утилиты для функциональной работы со списками

```

## Система обобщенных контейнеров

### Класс 1: Box<T> (Обобщенная коробка)
**Задача:** Универсальный контейнер для хранения одного объекта любого типа.

**Свойства:**
- `item` (private T) - хранимый объект

**Конструкторы:**
- `Box()` - пустая коробка
- `Box(T item)` - коробка с начальным значением

**Методы:**
- `put(T item)` - помещение объекта в коробку
- `get()` - извлечение объекта из коробки
- `getAsDouble()` - получение числового значения как double
- `isFull()`, `isEmpty()` - проверки состояния
- `containsNumber()` - проверка на числовое содержимое

**Особенности:** 
- Гарантирует, что в коробке может находиться только один объект
- Автоматическое освобождение коробки при извлечении
- Безопасная работа с типами через generics

**Демонстрация:**
```java
Box<Integer> intBox = new Box<>(42);
Box<String> stringBox = new Box<>("Hello");
intBox.get(); // Коробка становится пустой
```

## Система сравнения объектов

### Класс 2: NumberValue (Числовое значение)
**Задача:** Реализация сравнения числовых значений через интерфейс Comparable.

**Свойства:**
- `value` (private int) - числовое значение

**Конструкторы:**
- `NumberValue(int value)` - создание с указанным значением

**Методы:**
- `compare(NumberValue other)` - сравнение с другим объектом
- `getValue()` - получение значения
- `toString()` - строковое представление

**Особенности:**
- Возвращает -1 (меньше), 0 (равно), 1 (больше) при сравнении
- Позволяет сортировать и сравнивать числовые объекты

**Демонстрация:**
```java
NumberValue num1 = new NumberValue(10);
NumberValue num2 = new NumberValue(5);
int result = num1.compare(num2); // 1 (num1 > num2)
```

## Математические утилиты

### Класс 3: MathUtils (Математические операции)
**Задача:** Утилиты для работы с числовыми значениями в коробках.

**Методы:**
- `findMax(List<Box<? extends Number>> boxes)` - поиск максимума в списке коробок
- `findMax(Box<? extends Number>... boxes)` - поиск максимума в массиве коробок

**Особенности:**
- Работает с коробками, содержащими любые числовые типы (Integer, Double, Float)
- Автоматическое преобразование типов через doubleValue()
- Обработка пустых коробок и отсутствия числовых значений

**Демонстрация:**
```java
Box<Integer> box1 = new Box<>(42);
Box<Double> box2 = new Box<>(3.14);
double max = MathUtils.findMax(box1, box2); // 42.0
```

## Функциональные интерфейсы

### Интерфейс 4: Function<T, R> (Функциональное преобразование)
**Задача:** Преобразование значений одного типа в другой.

**Методы:**
- `R apply(T value)` - применение функции к значению

### Интерфейс 5: Predicate<T> (Условие фильтрации)
**Задача:** Проверка условия для значений.

**Методы:**
- `boolean test(T value)` - проверка условия

### Интерфейс 6: Reducer<T> (Свертка значений)
**Задача:** Комбинирование двух значений в одно.

**Методы:**
- `T reduce(T accumulator, T current)` - свертка значений

### Интерфейс 7: Collector<T, P> (Коллекционирование)
**Задача:** Создание пользовательских коллекций из элементов.

**Методы:**
- `P create()` - создание новой коллекции
- `void add(P collection, T item)` - добавление элемента

## Функциональные утилиты для списков

### Класс 8: ListUtils (Утилиты для работы со списками)
**Задача:** Реализация функциональных операций над списками.

**Методы:**
- `map(List<T> list, Function<T, R> function)` - преобразование каждого элемента
- `filter(List<T> list, Predicate<T> predicate)` - фильтрация по условию
- `reduce(List<T> list, Reducer<T> reducer, T defaultValue)` - свертка к одному значению
- `collect(List<T> list, Collector<T, P> collector)` - коллекционирование в пользовательскую структуру

**Демонстрация:**
```java
// Преобразование строк в их длины
List<Integer> lengths = ListUtils.map(strings, str -> str.length());

// Фильтрация положительных чисел
List<Integer> positives = ListUtils.filter(numbers, n -> n > 0);

// Сумма чисел
Integer sum = ListUtils.reduce(numbers, (a, b) -> a + b, 0);
```
